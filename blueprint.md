# **App Name**: Apna Partner

## Core Features:

- Secure OTP Login: Users authenticate via mobile OTP using Firebase Authentication to ensure a seamless and secure entry.
- Profile Builder: Users can create and manage a detailed profile including photo upload, personal information (name, age, gender, Jharkhand district from a dropdown), and a personalized bio.
- AI Bio Assistant: An AI tool to help users craft engaging and personalized profile bios, incorporating regional preferences or conversation starters.
- Intuitive Swipe System: A 'Tinder-like' swipe interface (right for like, left for pass) to discover potential profiles, leading to a match on mutual interest.
- District-Focused Filtering: Filter potential matches by specific districts within Jharkhand, allowing users to connect with individuals from areas like Chatra, Ranchi, or Hazaribagh.
- Real-time Match Chat: Matched users can engage in instant, real-time messaging, facilitating immediate connections and conversations.
- User Safety Controls: Functionality for users to report inappropriate profiles or block other users for enhanced safety and a respectful community environment.

## Style Guidelines:

- Primary color: Vibrant red-pink (#F20F67) evoking passion and energy, standing out effectively on lighter backgrounds.
- Background color: A very light, desaturated pink (#F7E1E9) that provides a soft, modern base for the vibrant elements, similar to a clean white but with a subtle warm undertone.
- Accent color: A bold fuchsia (#FF00CC) to highlight calls-to-action, key notifications, and interactive elements, providing strong contrast.
- Headline font: 'Playfair' (serif) for an elegant and fashionable aesthetic in titles and names. Body font: 'PT Sans' (humanist sans-serif) for clean readability in bios, chat messages, and general text.
- Custom, stylized icons inspired by the provided flame and heart motifs. Modern, sleek designs should align with the overall pink-red gradient theme and glassmorphism elements.
- Mobile-first, fully responsive UI with an emphasis on rounded cards for user profiles, promoting a friendly and contemporary feel. Integration of glassmorphism elements for depth and modern elegance.
- Smooth and engaging swipe animations for profile browsing, coupled with attractive transitions between pages to enhance user experience. Include a lively animated splash screen.
